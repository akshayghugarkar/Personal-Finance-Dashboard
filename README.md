# Personal Finance Dashboard - India

A modern, responsive personal finance dashboard designed for users in India with Indian Rupee (₹) formatting.

## Features

- 📊 Comprehensive financial overview with KPIs
- 📈 Income vs Expenses charts with time filtering
- 🥧 Expense breakdown by category
- 🎯 Savings goals tracking
- 💰 Budget monitoring
- 📝 Transaction management with search and filter
- 💡 Smart insights and forecasting
- 📱 Fully responsive design

## Local Development

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required!

## Customization

You can easily customize:
- Color scheme in `css/style.css`
- Transaction data in `js/script.js`
- Chart configurations in `js/script.js`
